git 创建分支 git branch branchName git 切换分支 git checkout branchName 组合 git checkout -b branchName

git 添加到暂存区 git add filePath git 提交到本地仓库 git commit -m "message" 组合 git commit --all 输入message

git 从当前分支的某次 提交切换分支 git checkout -b 71fa2e0 新的分支 从当前分支的71fa2e切换过来

git 放弃部分修改并切换分支 git checkout -f branchName git 储存修改（所有分支共用） git stash 查看储存 git stash list 从储存恢复最后一次存储 git stash pop

git 恢复删除的分支 git branch branchName 742dcf6（删除分支的散列值）
